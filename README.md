# RescueMonitor

RescueMonitor is a backend monitoring and uptime tracking tool designed to help developers monitor API endpoints, track performance, log failures, and maintain backend system stability.

## Features (Planned)
- User authentication
- Projects
- Endpoint monitoring
- Scheduled health checks
- Uptime tracking
- Response time tracking
- Failure logging
- Monitoring dashboard

## Project Structure
frontend/ - Dashboard UI
backend/ - API and monitoring engine
database/ - Schema and migrations
docs/ - Architecture and documentation
